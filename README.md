# Portfolio

> Minimal. Sharp. Fluid.

Personal portfolio built with **Next.js 14** (App Router) + **Framer Motion**.  
Black & gold — no noise, just craft.

## Stack

- **Framework** — Next.js 14 (App Router)
- **Animations** — Framer Motion
- **Styling** — Tailwind CSS
- **Déploiement** — Vercel
- **Domaine** — Porkbun / Namecheap
